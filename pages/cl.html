HiZollo 的更新日誌

beta 0.10.x

0.10.5
  • 修復了一些 bug
  • 新增以 ? 開頭的附加指令
    • 這些指令只有在 HiZollo 後臺伺服器中才能使用
  • 新增 z!lightsup 點燈遊戲
    • 在遊戲過程中可以輸入 answer 以顯示答案
    • 也可以輸入 stop 提前結束遊戲
  • 在 z!think 指令中新增其他表情
    • 你可以讓他隨機發送，也可以指定
    • 詳細使用方式可以使用 z!help think 查詢
0.10.4a
  • 修復了一些 bug
  • 修復了一些排版問題
  • 修復了部份時候會出現的格式問題
  • 新增 z!music 下的 again 指令
    • 會把當前正在播放的東西加入隊列最上方
    • 如果想要再聽一次，這樣或許方便了點
  • 校正回歸端午節
    • 絕對不是因為我們記錯日期，哈哈
0.10.4
  • 修復了一些 bug
  • 修復了一些排版問題
  • 新增 z!minecraft 指令以及其下的 crafting 指令
    • 可以搜尋 minecraft 中的物品的合成方式
    • 未來會增添更多 minecraft 相關指令
  • 因應投票結果，我們新增了 z!links，並移除了 z!invite 和 z!hzweb
    • 該指令會收錄 HiZollo 的相關連結
    • 包含機器人清單上的連結，請務必幫我們投下一票
  • 將 z!serverinfo 改名為 z!server
    • 因為原本的 z!server 已經併入 z!links
0.10.3a
  • 修復了一些 bug
  • 修改了子指令類的排版模式
  • 更改了 z!music play 的搜尋機制
    • 只會搜尋非網址格式的文字
  • 現在起，z!music play 可以抓取播放清單了
    • 當連結是播放清單格式時，會抓取播放清單的內容並加入隊列
    • 抓取清單可能會花上一些時間，請耐心等候
  • z!music skip 的功能更多了
    • 除了原有功能，也可用來移除隊列中的一串曲目
0.10.3
  • 修復了一些 bug
  • 修復了一些排版問題
  • 現在起，z!music play 可以直接搜尋了
    • 當你輸入的東西不是合法的網址時，就會被當作搜尋字詞
    • 可以直接在 Youtube 上搜尋並點播
  • 將 z!music playlist 及 z!music nowplaying 的排版重構
    • 有圖可以看，這樣是不是就不會太單調了
    • 也多了不少資訊
  • z!dice 可以透過輸入參數來取得特定骰子了
    • 使用 z!help dice 可以看到所有使用方法
  • 重新排版了 z!guildrop 顯示出來的清單
0.10.2
  • 修復了一些 bug
  • 修復了一些排版問題
  • 新增 z!music 下的 loop 指令
    • 可以讓機器人重複播放目前樂曲
    • 還有點不太大的小問題，望能儘速解決
  • 指令清單重新排版了
    • 新增「子指令」分類，將「功能」分類中有子指令的指令獨立出來
    • 這是為了避免「功能」分類太長候
  • 重新設計 z!dice
    • 骰子經過重新繪製，並使用表情符號方式傳送
0.10.1
  • 修復了一些 bug
  • 新增 osu! 指令
    • 使用 z!osu 可以看到目前有的功能
    • 未來將陸續添加
  • 新指令：z!deletemsg
    • 可以幫助你在一定時間後刪除訊息
0.10.0
  • 修復了一些 bug
  • 完整了整個音樂指令
    • 使用 z!music 可以看到所有東西
    • 目前已有近乎所有基本音樂功能
    • 剩下部分小功能會陸續補齊
