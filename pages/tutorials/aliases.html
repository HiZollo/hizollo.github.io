一般指令

當你在使用 z!help 查詢指令時，你會在某些指令的說明上看到「替代用法」： 這個替代用法就是指令的別名，通常會比原本的名稱短，讓你可以更快速、方便的使用 HiZollo 的指令，而不用輸入一長串的名字。舉例來說，這個 calc 指令，除了使用 z!calc 以外，你也能使用 z!c 來執行：

群組指令

群組指令跟群組名稱也有可能會有別名，而且都會列在指令群清單上，如下方： 可以看到，群組名稱 osu 的別名是 o，所以你也可以使用 z!o 來呼叫它；而其下的 user 指令有個替代用法 u，所以當你想使用 osu 的 user 指令時，總共可以有以下幾種方式：
  • z!osu user chocomint
  • z!osu u chocomint
  • z!o user chocomint
  • z!o u chocomint
這些用法都是合法的，出來的結果也都是一樣的。你可以在群組名稱使用別名，也能在群組指令的名稱使用別名，當然也能同時使用，這可以減低使用群組指令時，要輸入很多東西的麻煩。
